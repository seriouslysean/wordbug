An educational vocabulary site featuring words my son Jacob thinks are cool. This site showcases daily word definitions from Wordnik with pronunciation, etymology, and cross-references, serving as both a learning tool and historical reference.

🌐 **Live Site**: [wordbug.fyi](https://wordbug.fyi)

## About

WordBug is a father-son project where Jacob curates interesting vocabulary words, and I (Sean) build the site to showcase them. Each word comes with detailed definitions, pronunciation guides, etymology, and usage examples from Wordnik's comprehensive dictionary database.

## Features

- **Daily Word Display**: Curated vocabulary words with rich definitions
- **Historical Archive**: Browse all past words chronologically  
- **Cross-Reference Links**: Inline word references link to deeper exploration
- **Mobile-Friendly**: Responsive design works great on all devices
- **Fast & Accessible**: Static site optimized for speed and accessibility
